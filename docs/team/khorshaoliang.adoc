== Project: Blast!
Blast! is a desktop address book application developed with tutors and sales persons in mind The user interacts with it using a CLI, and it has a GUI created with JavaFX. It is written in Java, and more than 6 kLoC.

*Code contributed*: [https://github.com/KhorSL/main[Functional code]] [https://github.com/KhorSL/main[Test code]]
