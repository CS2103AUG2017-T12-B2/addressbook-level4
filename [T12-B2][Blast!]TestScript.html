1. Set up

  1. Download all documents (i.e. all the test files) uploaded on v1.5 release.

  2. Create a folder named ‘Blast’ and place the .jar file in

  3. Create a folder named ‘data’ at the root of ‘Blast’ folder

  4. Create a folder named ‘avatars’ at the root of ‘Blast’ folder

  5. Place the downloaded test image into avatars folder

  6. Place the downloaded test documents into data folder

  7. Launch the application
